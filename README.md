# LanSynK

**lansynK** is an tool, which allows you to, *contribute* in `real time` in your **OWN LAN**

### why?

- its works **offline**
- its **real time**
- no need to install big, things like... *git* and create a new account on, github just to... work on a project
- its small, but still affective
- fun to use, and we listen to you and, make *LanSynK* better

***

**lansynK** was orginally, made for *myself* and *my brother* to, contribute in `real time` without *installing* big *toolings*
and, for him to not *sign up for github and all* and, just work without a headache.

**And then...** I decided to, why not, release this for everyone to use?

and, now enjoy!
